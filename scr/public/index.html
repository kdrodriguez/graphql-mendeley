Introducción

La API GraphQL de Mendeley permite a la comunidad de desarrolladores implementar soluciones tecnológicas usando los servicios que brinda el gestor por medio del lenguaje de consultas GraphQL. Actualmente, Mendeley tiene una API Rest para brindar estos servicios, la API GraphQL usa estos servicios para generar nuevos servicios GraphQL, puede consultar más a detalle la API Rest en la página oficial de desarrollo de Mendeley.

Como usar

Para usar la API GraphQL de Mendeley, necesita disponer de una cuenta de usuario, si no cuenta con una puede revisar la página oficial de Mendeley para ver las opciones de registro. Puede crear consultas personalizadas usando la específicación GraphQL; para más información de GraphQL, visite su página oficial aquí. Descargue el manual de usuario de la API aquí

IDE GraphQL Playground

Haga consultas personalizadas con el IDE gráfico e interactivo GraphQL Playground . Escoja esta opción solo si tiene un token válido antes del tiempo de caducidad del mismo por que será redirigido directamente a Playground
