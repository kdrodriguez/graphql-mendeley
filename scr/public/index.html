Acerca

La API GraphQL de Mendeley permite a la comunidad de desarrolladores implementar soluciones tecnológicas usando los servicios que brinda el gestor por medio del lenguaje de consultas GraphQL. Actualmente, Mendeley tiene una API Rest para brindar estos servicios, la API GraphQL usa estos servicios para generar nuevos servicios GraphQL, puede consultar más a detalle la API Rest en la página oficial de desarrollo de Mendeley.

Como usar

Para usar la API GraphQL de Mendeley, necesita disponer de una cuenta de usuario, si no cuenta con una puede revisar la página oficial de Mendeley para ver las opciones de registro. Puede crear consultas personalizadas usando la específicación GraphQL; para más información de GraphQL, visite su página oficial aquí. Descargue el manual de usu de la API aquí
